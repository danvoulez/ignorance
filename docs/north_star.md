# North Star

*Draft.*

One-page compass: what we are building and why.

---

**Mission:** To show that ignorance can and must take form — that the unknown is not a gap to be filled by silence or fabrication, but an object to be instantiated, governed, and carried until resolution is legitimate.

**Core form:** The canonical cell (who, did, this, when, confirmed by, if ok, if doubt, if not, status) as the smallest unit that carries action, causality, evidence, branching, and state across scales.

**Principle:** A trustworthy system is not one that always knows. It is one that knows how to give form to what it does not yet know, before the pressure to continue forces it to lie.

**Organism:** Book, spec, formal, and code co-evolve. No layer is mere documentation of another.
