# Docs — Living Research Layer

This folder holds the **living research** layer: north star, one-page thesis, glossary, language decisions, chapter status, open questions, rejected paths, future work, and decision log. It is the place where the organism’s choices are recorded and where terminology is fixed before it flows into book, spec, and code.

---

## Files

| File | Purpose |
|------|---------|
| [north_star.md](north_star.md) | One-page compass: what we are building and why |
| [one_page_thesis.md](one_page_thesis.md) | Compressed thesis for quick reference |
| [glossary.md](glossary.md) | Canonical definitions; source of truth for terms |
| [language_decisions.md](language_decisions.md) | Naming and phrasing choices (e.g. Ghost vs placeholder) |
| [chapter_status.md](chapter_status.md) | Draft / review / done status per book chapter |
| [open_questions.md](open_questions.md) | Live list of unresolved questions |
| [rejected_paths.md](rejected_paths.md) | Paths we chose not to take, with brief reason |
| [future_work.md](future_work.md) | Work deferred or out of scope for current waves |
| [decision_log.md](decision_log.md) | Material decisions (structure, tooling, scope) with date and rationale |

---

## Rules

- **Glossary wins.** New terms are proposed here before use in book, spec, or code.
- **Decision log is append-only.** Entries are dated; reversals are new entries with reference to the old one.
- **Open questions** are triaged: answer, defer to future_work, or reject and record in rejected_paths.
